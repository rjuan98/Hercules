# Hércules

Assistente financeiro e fiscal humano.

## Como rodar

```bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# ou
.venv\Scripts\activate     # Windows

pip install -r requirements.txt
python app.py
```

Abra `http://127.0.0.1:5000`.

## O que tem nesta versão

- login e cadastro;
- perfis: PF, MEI, lojista e híbrido;
- home personalizada;
- saldo, projeção e "quanto posso gastar hoje";
- notas fiscais;
- movimentações financeiras;
- metas ilimitadas;
- compromissos e vencimentos;
- clientes e serviços;
- painel de negócio;
- exportação de IR em CSV.
